下側のボーダーが上

見出し1

見出し1

コード

HTML
<!-- 背景画像を使用（左側のボーダーが上） -->
<h1 class="el-heading-borderLeft el-heading-borderBottom_bg">見出し1</h1>

<!-- 外側にdivを使用（左側のボーダーが上） -->
<div class="el-heading-wrap_borderLeft">
<h1 class="el-heading-borderBottom">見出し1</h1>
<!-- el-heading-wrap_borderLeft --></div>

<!-- 背景画像を使用（下側のボーダーが上） -->
<h1 class="el-heading-borderLeft_bg el-heading-borderBottom">見出し1</h1>

<!-- 外側にdivを使用（下側のボーダーが上） -->
<div class="el-heading-wrap_borderBottom">
<h1 class="el-heading-borderLeft">見出し1</h1>
<!-- el-heading-wrap_borderBottom --></div>
CSS(common.css)
h1, h2, h3, h4, h5, h6 {
  margin: 20px 0;
}

/* 見出し（左側ボーダー） */
.el-heading-borderLeft {
  padding-left: 10px;
  border-left: solid 4px #009944;
}

/* 見出し（下側ボーダー） */
.el-heading-borderBottom {
  border-bottom: solid 2px #CCC;
}

/* 見出し（L字ボーダー） */
/* 背景画像を使用（左側のボーダーが上） */
.el-heading-borderBottom_bg {
  padding-bottom: 2px;
  background: url(..//images/bg-heding-bd-bottom.png) left bottom repeat-x;
}

/* 外側にdivを使用（左側のボーダーが上） */
.el-heading-wrap_borderLeft {
  border-left: solid 4px #009944;
}
.el-heading-wrap_borderLeft .el-heading-borderBottom {
  padding-left: 10px;
}

/* 背景画像を使用（下側のボーダーが上） */
.el-heading-borderLeft_bg {
  padding-left: 14px;
  background: url(..//images/bg-heding-bd-left.png) left top repeat-y;
}

/* 外側にdivを使用（下側のボーダーが上） */
.el-heading-wrap_borderBottom {
  margin-bottom: 20px;
  border-bottom: solid 2px #CCC;
}
.el-heading-wrap_borderBottom .el-heading-borderLeft {
  margin-bottom: 0;
}

注意

一つの要素へ左側と下側のボーダーを適用した場合、コーナー部分は斜めに処理されます。またその処理方法もブラウザ間で差異が出ます。
