作例

こちらのボックスを作成しましょう。

border-style:プロパティを使用

プラン名

テキストテキストテキストテキストテキストテキストテキストテキストテキスト

価格 nn,nnn円

border-style:プロパティでは線の太さを個別に調整することができません。ではこちらの方法ではどうでしょう。

divを二重にする

プラン名

テキストテキストテキストテキストテキストテキストテキストテキストテキスト

価格 nn,nnn円

コード

HTML
<!-- 二重線ボックス（border-style:プロパティを使用） -->
<div class="el-box03 el-boxDoubleFrame_css el-boxFrame el-marginBottom20">
<p class="mod-box-products_title">プラン名</p>
<p class="mod-box-products_text">テキストテキストテキストテキストテキストテキストテキストテキストテキスト</p>
<p class="mod-box-products_price">価格 nn,nnn円</p>
<!-- /el-box03 --></div>

<!-- 二重線ボックス（divを二重にする） -->
<div class="el-box03 el-boxDoubleFrame el-boxFrame el-marginBottom20">
<div class="el-box03 el-boxFrame">
<p class="mod-box-products_title">プラン名</p>
<p class="mod-box-products_text">テキストテキストテキストテキストテキストテキストテキストテキストテキスト</p>
<p class="mod-box-products_price">価格 nn,nnn円</p>
<!-- /el-box03 --></div>
<!-- /el-boxDoubleFrame --></div>
CSS(common.css)
/* ボックスモデル共通 */
.el-box03 {
  padding: 13px;
}

.el-boxFrame {
  border-style: solid;
  border-width: 1px;
}

/* ボックス3（青） */
.el-box03 {
  background-color: #eaf6ff;
}

.el-box03.el-boxFrame{
  border-color: #0068b7;
}

/* 二重線ボックス（border-style:プロパティを使用） */
.el-boxDoubleFrame_css {
  border-style: double;
  border-width: 5px;
}

/* 二重線ボックス（divを二重にする）の外側 */
.el-boxDoubleFrame {
  padding: 1px;
  border-style: solid;
  border-width: 3px;
}

/* ボックス内モジュール */
.mod-box-products_title {
  margin: 0 0 5px 0;
  font-weight: bold;
}

.mod-box-products_text {
  margin: 0;
}

.mod-box-products_price {
  margin: 15px 0 0;
}

補足

コードを見ると、デザインのためにdivが二重になっています。古いブラウザを考慮するなどやむを得ない場合もありますが、装飾のためだけにdiv要素を増やすのはなるべく避けたいところです。次の記事以降で一つのdivのみで作例通りの二重線を作成する方法を紹介していきます。
