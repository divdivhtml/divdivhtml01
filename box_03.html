box-shadowプロパティの注意

複数のbox-shadowプロパティが1つの要素で同時に存在した場合、もっとも優先度の高いプロパティしか適用されません。こちらの例では、インナーボックスのプロパティが優先され、ドロップシャドウのプロパティは打ち消されています。

CSS(common.css)

.el-box03 {
  padding: 13px;
}

/* ボックス3（青） */
.el-box03 {
  background-color: #eaf6ff;
}

/* セレクタ優先度が同等で、先に記述されているため無効となる */
.el-boxShadow {
  box-shadow: 1px 1px 4px 1px rgba(0, 0, 0, 0.25);
}

/* セレクタ優先度が同等で、後で記述されているため有効となる */
.el-boxShadowInset {
  box-shadow: 0 0 8px 1px rgba(0, 0, 0, 0.25) inset;
}

}
