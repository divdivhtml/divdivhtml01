コード

HTML（floatを利用）
<div class="el-heading-text-H1_float">
<h1 class="el-heading-text-h_float">見出し1</h1>
<p class="el-heading-text_float"><a href="#DUMMY">テキストリンク</a></p>
<!-- /el-heading-text-H1 --></div>

<div class="el-heading-text-H2_float">
<h2 class="el-heading-text-h_float">見出し2</h2>
<p class="el-heading-text_float"><a href="#DUMMY">テキストリンク</a></p>
<!-- /el-heading-text-H2 --></div>

<div class="el-heading-text-H3_float">
<h3 class="el-heading-text-h_float">見出し3</h3>
<p class="el-heading-text_float"><a href="#DUMMY">テキストリンク</a></p>
<!-- /el-heading-text-H3 --></div>
CSS(common.css)（floatを利用）
h1, h2, h3, h4, h5, h6 {
  margin: 20px 0;
}

/* 見出し（右端にテキストリンク） */
/* clearfix */
.el-heading-text-H1_float:after,
.el-heading-text-H2_float:after,
.el-heading-text-H3_float:after{
  content: "";
  clear: both;
  display: table;
}

/* IE6/7に対応したclearfix（IE6/7に対応しない場合は記述不要） */
.el-heading-text-H1_float,
.el-heading-text-H2_float,
.el-heading-text-H3_float{
  *zoom: 1;
}

/* 見出し（右端にテキストリンク） */
.el-heading-text-H1_float,
.el-heading-text-H2_float,
.el-heading-text-H3_float {
  margin: 20px 0;
  border-bottom: solid 2px #F2F2F2;
}

.el-heading-text-h_float {
  float: left;
  margin: 0;
}

.el-heading-text_float {
  float: right;
  margin: 0;
}

注意

文字があふれた場合、段落ちして表示されます。また、テキストリンクにマージンやパディングを設定していた場合はその分アキが生じます。
